 Project Overview

This project collects publicly available job postings for Data Analyst roles from an online job portal.
The following fields are extracted:

Job Title

Company Name

Location

Salary (if available)

Job Summary / Description

Skills (extracted from description using regex)

Job Link

The scraped data is then cleaned, structured, analyzed, and visualized.
